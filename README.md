# LockBox

LockBox is a secure password manager and vault built with Python, Kivy, and KivyMD.  
It allows you to safely store and manage passwords, with features such as encryption, password generation, and more.

## Features
- AES-GCM encryption for secure storage
- Easy-to-use GUI with Kivy/KivyMD
- Export passwords to `.xlsx` files
- Clipboard management for secure copy/paste
